# Git

#### Skip local changes

```text
git update-index --skip-worktree <file_name>
git update-index --no-skip-worktree <file_name>
```

#### Get commit hashes from

From a tag: `git rev-parse TAG_NAME^{}`

From head: `git rev-parse HEAD`
